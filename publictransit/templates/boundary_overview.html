A Ref GSS code is a set of letters and numbers used by the UK's Office for National Statistics to identify geographical areas for statistical purposes. These codes are used to distinguish between different levels of geographic areas, such as regions, local authorities, and electoral wards.

An ISO 3166-2 code is a standardized code used to identify the administrative subdivisions of the country. The code consists of the country code "GB" (for United Kingdom) followed by a hyphen and a code for the subdivision, such as "ISL" for the London Borough of Islington or "LDN" for the City of London.
